- Use `!bs`-(Element bootstrap)<br>
Example:`!bs`-arlert

- Use `!crt`-(Snippet Simple Template)<br>
Example:`!crt`-login

- `!bs-start-cdn`   -> for starter Template with CDN CSS Bootstrap,CDN Jquery JS ,CDN Poppers JS
